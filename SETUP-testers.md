# إعداد صفحة المختبرين الداخليين (حفظ البيانات في Google Sheet)

صفحة `testers.html` تجمع بيانات المختبرين (الاسم، البريد، نوع الجهاز) وتُرسلها
إلى **Google Apps Script Web App** الذي يحفظها تلقائياً في **Google Sheet**.

اتبع الخطوات التالية مرة واحدة فقط (٥ دقائق):

---

## 1) أنشئ جدول جوجل شيت

1. افتح <https://sheets.google.com> وأنشئ جدولاً جديداً.
2. سمِّه مثلاً: **مختبرو سُمو**.
3. لا حاجة لإضافة أعمدة يدوياً؛ سيُنشئها السكربت تلقائياً.

---

## 2) الصق كود Apps Script

1. من داخل الجدول: **الإضافات (Extensions) ← Apps Script**.
2. احذف أي كود موجود، والصق كامل محتوى الملف **`google-apps-script.gs`**.
3. اضغط **حفظ** (أيقونة القرص).

---

## 3) انشر السكربت كتطبيق ويب

1. أعلى اليمين: **Deploy ← New deployment**.
2. اضغط أيقونة الترس ⚙️ بجانب "Select type" واختر **Web app**.
3. اضبط الإعدادات كالتالي:
   - **Description:** `Sumo testers`
   - **Execute as:** `Me` (بريدك)
   - **Who has access:** `Anyone`  ← مهم جداً حتى يعمل النموذج للجميع.
4. اضغط **Deploy**، ثم **Authorize access** ووافق على الأذونات
   (قد تظهر شاشة تحذير من جوجل → **Advanced ← Go to … (unsafe)** → **Allow**).
5. انسخ **Web app URL** (ينتهي بـ `/exec`).

---

## 4) اربط الرابط بالصفحة

1. افتح ملف **`testers.js`**.
2. في السطر الأول تقريباً، استبدل القيمة:

   ```js
   const ENDPOINT = 'ضع_هنا_رابط_تطبيق_Apps_Script';
   ```

   بالرابط الذي نسخته:

   ```js
   const ENDPOINT = 'https://script.google.com/macros/s/AKfy..../exec';
   ```

3. احفظ الملف وارفعه (نفس مجلد الموقع).

---

## 5) جرّب

- افتح `testers.html` في المتصفح، واملأ النموذج، واضغط **سجّلني كمختبِر**.
- ستظهر رسالة نجاح، وسيُضاف صف جديد في الجدول يحتوي:
  **التاريخ · الاسم · البريد الإلكتروني · نوع الجهاز · المصدر**.

> **ملاحظة:** إذا عدّلت كود `google-apps-script.gs` لاحقاً، أعد النشر عبر
> **Deploy ← Manage deployments ← تعديل (✏️) ← New version ← Deploy**
> حتى تسري التغييرات (الرابط يبقى نفسه).

---

## رابط الصفحة

المسار الخاص بصفحة التسجيل هو:

```
https://<اسم-موقعك>/testers.html
```

يمكنك مشاركة هذا الرابط مباشرةً مع المختبرين، أو ربطه من زر في الصفحة الرئيسية.
الصفحة مضبوطة على `noindex` كي لا تظهر في نتائج البحث.
